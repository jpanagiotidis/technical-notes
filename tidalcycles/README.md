# TidalCycles

## Start
1. Start SuperCollider
1. Start SuperCollider SuperDirt synthesizer 
```
SuperDirt.start
```
1. Start Visual Studio Code
1. Create a file with .tidal extension
1. Play sound
```
d1 $ sound "bd sn"
```
1. Hit SHIFT+ENTER
1. Stop sound by CTR+ALT+H

## Play
### Sinlge line
```SHIFT + ENTER```
### Multiple lines
```CTRL + ENTER```
