# Mesh Modeling

## Switch Mode (from object mode to edit mode)
* press tab

## Translate
* press g

## Rotate
* press r

## Scale
* press s

## Open Vertex Menu
* press ctrl + v

## Open Edge Menu
* press ctrl + e

## Open Face Menu
* press ctrl + f

## Join Objects
* go to Object Mode
* press ctrl + j

## Seperate Objects
* go to Edit Mode
* make selection
* press p

## Add vertex
* go to edit mode
* select a vertex
* press ctrl + left mouse button

## Add Edge
* go to edit mode
* select 2 vertices
* press f

## Add Face
* go to edit mode
* select 3 or more vertices
* press f
