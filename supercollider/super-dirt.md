# SuperDirt

## Installation
```Quarks.checkForUpdates({Quarks.install("SuperDirt", "v1.1.3"); thisProcess.recompile()})```

## Basics

### Start
```SuperDirt.start```

### Stop
```SuperDirt.stop```
