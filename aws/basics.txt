## Get SSH access
  1) add IP
    a) go to "Network & Security"/"security groups"
    b) tick and edit security group
    c) add rule with SSH ip/32
  2) create key
    a) go to "Network & Security"/"Key Pairs"
    b) click "Create Key Pair"
  3) set key permissions
    a) chmod 400 /path/to/key.pem
  4) start ssh
    a) ssh -i /path/to/key.pem ec2-user@host
