## Create Project
  1) run: 
      cordova create [project_directory] [reverse_domain-style_identifier] [project_title]
    example: 
      cordova create hello com.example.hello HelloWorld

## Add platforms
  1) run:
      cordova platform add [platform_name]
    example:
      cordova platform add android

## List added platforms
  1) cordova platforms ls

## Remove platforms
  1) run:
    cordova platform rm [platform_name]
    (or cordova platform remove [platform_name])

## Build project
  1) cordova build android

## Run project on device
  1) cordova run android

## Create debug release
  1) cordova build android --release
  2) go to android folder
  3) ant debug
