## Ubuntu 14.04
	## Sublime 2
		sudo add-apt-repository ppa:webupd8team/sublime-text-2
		sudo apt-get update
		sudo apt-get install sublime-text
	## Sublime 3
		sudo add-apt-repository -y ppa:webupd8team/sublime-text-3
		sudo apt-get update
		sudo apt-get install sublime-text-installer
