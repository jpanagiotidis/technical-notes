# Haskell Command Line

## GHCI

### Value Type
```
ghci> :t {VALUE}
```

### Name Info
```
ghci> :i {NAME}
```

### Type kind
```
ghci> :k {TYPE}
```

## Compile program
```
ghc --make {FILENAME}
```

## Run compiled program
```
./{FILENAME}
```

### Input redirection
```
./{FILENAME} < {INPUT_FILE}
```
