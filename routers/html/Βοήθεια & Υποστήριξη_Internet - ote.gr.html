        Port Forwarding

        Για τη διαδικασία του Port Forwarding, ακολουθήστε τη διαδικασία παρακάτω:

        (Εάν η πόρτα που θέλετε να ανοίξετε είναι TCP: 1 μέχρι και 1023, 1433, 3389 και 5000.UDP:161, 135 μέχρι και 139, και 1434 τότε θα πρέπει να απενεργοποιηθεί το φίλτρο αυξημένης προστασίας . Σε αντίθετη περίπτωση, προχωρήστε στο δεύτερο βήμα).


        Βήμα πρώτο (firststep):

        Απενεργοποιήστε (disable) το: Επιθυμία αυξημένης προστασίας από κακόβουλες "επιθέσεις".
        Μεταφερθείτε (goto) στο https://www.ote.gr/web/guest/help-and-support/internet/security/-/support/article/41638980.
        Kάντε αριστερό (leftclickon) κλικ στο Έλεγχος Βαθμού Αυξημένης Προστασίας.

        i) Aκολουθήστε τα βήματα (follow the steps in order to disable the filter of OTE) για να απενεργοποιήσετε το φίλτρο: Επιθυμία αυξημένης προστασίας από κακόβουλες "επιθέσεις".

        ii) Eπιλέξτε (choose) ΟΧΙ. Πατήστε Επιβεβαίωση. 






        iii) Kάντε on / off το modem/router για να πάρει την αλλαγή (in order to accept the change of OTEs filter).

        Ανοίξτε έναν browser (π.χ. Internet Explorer, Firefox ή Chrome κτλ.)

        Πληκτρολογήστε (go to) πάνω στην διεύθυνση (address bar): 192.168.1.1










        Βάλτε (type) το Device Password του router και πατήστε το Log In.











        Θα ανοίξει η αρχική σελίδα του modem / router. Βλέπετε ότι το εικονίδιο Internet είναι τσεκαρισμένο, που σημαίνει ότι ο router είναι συνδεδεμένος στο Internet. Πάνω πάνω, κάντε κλικ στο Internet για να δείτε τις ρυθμίσεις.









        Επιλέξτε το δεύετρο μενού Features και μέσα σε αυτό, επιλέξτε το Port Forwarding.










        Σημαντικό: Tικάρετε το Enable, βλέπετε παρακάτω συμπληρωμένη την καρτέλα.

        Στο πεδίο Name: πληκτρολογήστε ένα όνομα για την θύρα (insert a name) (π.χ. Camera Garaz #4).

        Επιλέξτε στο πεδίο Protocol: αν θα ανοίξει είτε τις TCP, είτε τις UDP.

        Προσοχή: Στο LAN Host IP Address εισάγετε την εσωτερική (internal) IP (π.χ. 192.168.1.22).

        Στα πεδία WAN Host Start IP Address και WAN Host End IP Address πληκτρολογήστε 0.0.0.0.

        Στο WAN Start Port και στο WAN End Port βάζετε την θύρα ή θύρες (port) που θα ανοίξετε (open).

        Στο LAN Host Start Port και στο LAN Host End Port βάζετε την θύρα ή θύρες (port) που θα ανοίξετε (open)        .
